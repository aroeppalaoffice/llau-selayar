💬
Setup WhatsApp — Fonnte Gateway
Gateway WA Indonesia, mulai gratis
1
Daftar di Fonnte
Buka fonnte.com → daftar akun → verifikasi nomor WhatsApp yang akan digunakan untuk kirim laporan (bisa nomor kantor atau nomor kamu).
2
Dapatkan Token API
Di dashboard Fonnte → menu Device → salin token yang tersedia. Token ini yang dimasukkan ke Apps Script dan form ini.
3
Update Apps Script
Di Apps Script, pada bagian CONFIG, ganti:
WA_TOKEN : "token_fonnte_kamu"
WA_KAKAN : "628nomor_kepala_kantor"
Simpan → Deploy ulang (versi baru).
4
Isi token di form ini
Klik Konfigurasi di pojok kanan atas → isi Token Fonnte dan Nomor Kepala Kantor → Simpan. Notifikasi WA akan aktif otomatis setiap kali data disimpan.
